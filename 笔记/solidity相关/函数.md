# 函数

## 四种可见性修饰符
### **1. public**
公开，人人都能访问

比external费gas

```solidity
function getData() public view returns (uint) {
    return data;
}
```

### **2. external 外部函数**
由地址和函数签名组成，可以通过外部函数调用传递和返回

内部调用除非用this

比public省gas

```solidity
function sayHi() external pure returns (string memory) {
    return "Hi!";
}

// 合约内部调用时需要：this.sayHi();
```

### **3. internal 内部函数**
内部函数,只能间接调用

默认情况下,函数类型是内部的

比如可以设一个external函数,通过这个函数来调用internal函数

```solidity
function iroha() internal{
    a=a-1;
}
function illit() external{
    iroha();
}
```

这里illit是外部合约，iroha是内部合约，通过illit来调用iroha（夹带私货^ ^）

### **4. private**
私有,只能在当前合约本身内部使用
