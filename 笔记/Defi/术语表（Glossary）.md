# 术语表（Glossary）

# Automated market maker（自动化做市商 / AMM）

自动化做市商是一种部署在以太坊上的智能合约，持有链上流动性储备。用户可以对这些储备进行交易，交易价格由自动化做市公式设定。

# Constant product formula（常数乘积公式)

Uniswap 使用的自动化做市算法。见 `x * y = k`。

# ERC20（ERC-20 代币）

ERC-20 代币是以太坊上的可替代（同质）代币。Uniswap 支持所有标准的 ERC-20 实现。

# Factory（工厂合约）

一个智能合约，用于为任意 ERC-20 / ERC-20 交易对部署一个**唯一**的智能合约实例。

# Pair（交易对合约）

由 Uniswap V2 Factory 部署的智能合约，允许两个 ERC-20 代币之间进行交易。

# Pool / 流动性池

在某个交易对（pair）内，所有流动性提供者（LP）对代币的注入被汇聚为该交易对的流动性池。

# Liquidity provider / LP（流动性提供者）

向某个交易对的流动性池存入等值的两种 ERC-20 代币的参与者。流动性提供者承担价格风险，并通过手续费获得补偿。

# Mid price（中间价 / 中点价）

某一时刻用户买入和卖出代币的价格之间的价位。在 Uniswap 中通常表示为两种 ERC-20 储备的比率（即储备间的比值）。

# Price impact（价格影响）

交易的中间价与其实际执行价格之间的差异。

# Slippage（滑点）

从交易提交到交易执行期间，交易对价格发生的变动幅度（即交易最终成交价相对于期望价的偏差）。

# Core（核心）

对 Uniswap 存在至关重要的智能合约集合。若要升级到新的核心版本，通常需要进行流动性迁移（liquidity migration）。

# Periphery（外围）

有用但非必须的外部智能合约。可以随时部署新的外围合约而无需迁移池内流动性。

# Flash swap（闪电交换）

一种在支付前就使用所购买代币的交易（即在同一笔交易内先借用代币并在交易结束前偿还或以其他方式结清）。

# x \* y = k

常数乘积公式。

# Invariant（不变量）

常数乘积公式中的 `k` 值（即 `x * y` 的固定乘积）
