# 动态数组

## 定义

在声明时不指定数组的长度,用`T[]`的格式声明，其中`T`是元素的类型

```solidity
uint[] array1;
bytes1[] array2;
address[] array3;
bytes array4;
```

这里这个`bytes`也是数组,他不用加`[]`

写法上注意,`byte[]`不对,`bytes1[]`**对**,但是没有`bytes`省gas

## 创建规则

1. 对于`memory`**修饰的动态数组**，用new来创建,但必须声明长度,且声明后不能修改

```solidity
uint[] memory array3 = new uint[](5);
bytes memory array4 = new bytes(9);
```

括号里那个5和9就是数组的长度,数组里边的每一个元素的type都以第一个元素为准

2. 动态数组需要一个个赋值

```solidity
uint[] memory x = new uint[](3);
x[0] = 1;
x[1] = 3;
x[2] = 4;
```

## length

所有数组都有 `.length` 属性

对于 `memory` 数组来说,创建时需要声明长度,而且创建后长度就**不能改变**，\*\*不能使用 **`push`** 或 \*\*`pop`。

对于 `storage` 动态数组来说，长度是可以改变的,就是可以使用 `push` 和 `pop`

## push()

只能用于`storage`动态数组

`push()` 不带参数，会在末尾<font style="background-color:#f3bb2f;">添加</font>一个**默认值元素**（`uint` 是 0，`bool` 是 false）

它**返回的是新添加元素的引用**,可以继续对它赋值

## push(x)

只能用于`storage`动态数组

在数组末尾<font style="background-color:#f3bb2f;">添加</font>一个`x`元素

## pop()

只能用于`storage`动态数组

<font style="background-color:#f3bb2f;">删除</font>数组最后一个元素,数组长度缩短
